# Chapter 01: JavaScript Basics

This folder contains introductory JavaScript examples to help you get started with the fundamentals.

## Files Overview

### 01_Basics.js
- Demonstrates basic `console.log()` usage
- Shows how to access `process.env` to inspect environment variables
- Includes commented examples for accessing `process.platform`, `process.version`, `process.arch`, and `process.cwd()`

### 02_JS.js
- Introduces variable declaration using `let`
- Demonstrates `for` loops for iteration
- Shows how to define and call a simple function (`print`)

### 03_JS_Verify_Setup.js
- Useful for verifying your Node.js environment setup
- Logs `process.platform`, `process.arch`, and `process.version`

### 04_HotCode.js
- Demonstrates a function with parameters (`add(a, b)`)
- Uses a `for` loop to perform repeated calculations
- Logs iteration index and computed results

## How to Run

You can run any of these files using Node.js:

```bash
node 01_Basics.js
node 02_JS.js
node 03_JS_Verify_Setup.js
node 04_HotCode.js
```

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine
